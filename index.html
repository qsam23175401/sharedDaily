安心鎖設定

啟用安心鎖後，每次開啟APP時都必須輸入密碼，並且日記被破譯難度大幅提高。

注意！密碼忘記就再也找不回日記了！
若要關閉，請輸入密碼後按下「關閉安心鎖」按鈕。

設定 AI 如何模仿您的風格來總結當日小記。

*此欄內容清空則使用上方選項。

如何取得 Gemini API Key?

  1. 前往 Google AI Studio
  2. 使用您的 Google 帳號登入
  3. 左下角 Get API key
  4. 點擊 Create API key 按鈕
  5. 隨意創建一個專案即可生成API Key
  6. 複製後請妥善保管，不要洩漏給他人

作者的話

感謝您使用這款心情日記本！
希望能讓您記錄生活變得更輕鬆、也更溫暖。
日記內容會簡單加密後儲存在個人裝置上，不會上傳任何伺服器，並可供離線使用。

您的隱私我很在意，但我能在程式碼中做的有限，以下安全措施請自行斟酌：
　1. 請勿使用您不信任來源的擴充套件。即使是擴充商店中，經過安全驗證的套件，也可能會竊取您當下網頁的隱密資料。
　2. 設定中有安心鎖加密模式，如果裝置與家人共用，可以打開；並且安心鎖開啟後，即使資料被竊，也無法被輕易破譯。
　3. gemini api key 可以於google cloud中設定允許的網域，以確保安全。
　　- gcloud網址：https://console.cloud.google.com/apis
　　- 登入後，左上角選擇您在AI studio中建立的專案。
　　- 選擇「憑證」->「動作」->「編輯 API 金鑰」
　　- 繼續在「應用程式限制」中，選擇「網站」->「add」(2次)
　　- 允許的網域要設定兩個，設定後要「儲存」!!

# sharedDaily - 心情日記本

**sharedDaily** 是一個極簡、注重隱私且結合 AI 功能的本地端日記應用程式。它專為希望快速記錄生活點滴的使用者而設計。

## 🌟 核心特色

-   **🔒 隱私至上**：所有日記資料僅儲存在您裝置的瀏覽器（localStorage）中。我們沒有資料庫，也不會將您的私密文字上傳到任何伺服器。
-   **🎙️ AI 語音解析**：整合 Google Gemini API，可以直接口述生活瑣事至日常小記，由 AI 自動解析出「時間點」與「事件內容」，讓晚上寫日記更輕鬆。
-   **✨ AI 日記總結**：寫下一條條的小記後，點擊「幫我寫日記」，AI 會模仿您的口吻（可自訂人稱、語氣與關注重點）將碎片化的紀錄轉化為精彩的日記主文。
-   **🎨 心情色彩**：自訂心情標籤，用顏色索引您的每一天，回顧心情脈絡一目了然。
-   **📱 PWA 支援**：支援「加入主畫面」，像原生 App 一樣流暢，且支援離線瀏覽已存儲的日記。

## 🚀 快速開始

### 1. 取得 API 金鑰
為了使用 AI 功能（語音與總結），您需要一個免費的 Gemini API Key：
1. 前往 [Google AI Studio](https://aistudio.google.com/)。
2. 登入 Google 帳號後，點擊 **Get API key**。
3. 建立新專案並複製 API Key。

### 2. 設定專案
1. 開啟 [sharedDaily 網頁](https://qsam23175401.github.io/sharedDaily/)。
2. 點擊導覽列最右側的 **設定 (齒輪圖示)**。
3. 貼上您的 API Key 並儲存。
    - *註：Key 僅會保存在您的瀏覽器中。*

### 3. 安裝到手機
-   **iOS**: 使用 Safari 開啟 -> 點選「分享」 -> **加入主畫面**。
-   **Android**: 點選瀏覽器選單 -> **安裝應用程式** 或 **加入主畫面**。
